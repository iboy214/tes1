Program Keahlian Tersedia

1. TKJ

Bidang Keahlian : jaringan komputer dan aplikasinya

Program Keahlian : Program keahlian dalam teknik komputer jaringan biasanya mencakup berbagai mata pelajaran yang meliputi jaringan komputer, administrasi sistem, keamanan jaringan, protokol komunikasi, manajemen jaringan, pemrograman, dan konsep dasar teknologi informasi. Program-program ini bertujuan untuk mempersiapkan siswa atau mahasiswa dengan pengetahuan dan keterampilan yang diperlukan untuk bekerja di bidang teknologi informasi, khususnya dalam perencanaan, pengelolaan, dan pemeliharaan jaringankomputer.

Konsentrasi Keahlian : 1. pemrograman,
2. Jaringan Komputer,
3. Sistem Operasi,
4. Pemeliharaan Perangkat Keras.

2. OTKP

Bidang Keahlian : Bisnis dan Manajemen
Program Keahlian : Penggunaan Perangkat Lunak Kantor: Mahasiswa belajar untuk mengoperasikan dan menguasai berbagai perangkat lunak perkantoran seperti pengolah kata (misalnya Microsoft Word, Google Docs), spreadsheet (seperti Microsoft Excel, Google Sheets), presentasi (seperti Microsoft PowerPoint, Google Slides), dan perangkat lunak administrasi kantor lainnya.
Konsentrasi Keahlian : 1. Pengelolaan Perkantoran Elektronik,
2. Administrasi Perkantoran,

3. Otomatisasi Perkantoran,

4. Komunikasi Bisnis.

3. KEPERAWATAN

Bidang Keahlian : jurusan yang akan mendidikmu menjadi seorang tenaga kesehatan profesional.
1. Program Keahlian : Memahami dan memberikan perawatan yang holistik kepada pasien, termasuk penilaian kondisi kesehatan, perencanaan perawatan, pelaksanaan intervensi, dan evaluasi hasil perawatan.
Konsentrasi Keahlian : 1. Keperawatan Anak (Pediatric Nursing)
2.Keperawatan Dewasa (Adult Nursing)
3. Keperawatan Lansia (Gerontological Nursing)
4. Keperawatan Gawat Darurat (Emergency Nursing)

4. PEKSOS

Bidang Keahlian : Pekerja Sosial Generalis
Program Keahlian : melaksanakan proses pertolongan kepada klien atau penerima pelayanan, banyak peran Peksos profesional yang harus dilakukan.
